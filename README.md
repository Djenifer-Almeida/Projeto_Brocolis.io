Este projeto é uma aplicação web estática (frontend-only).
O objetivo central do sistema é disponibilizar uma plataforma simples e intuitiva para que o cliente consiga gerenciar
seus produtos — permitindo cadastrar, listar, visualizar e organizar itens de forma autônoma.
